AsalSI WebGIS

Pemetaan asal mahasiswa Sistem Informasi Universitas Andalas

Platform analisis spasial akademik untuk membaca persebaran asal sekolah, jalur masuk, angkatan, dan pola geografis mahasiswa secara ringkas, visual, dan mudah ditindaklanjuti.

Kapabilitas

Dirancang untuk kebutuhan analisis spasial akademik

Modul utama AsalSI membantu admin mengelola data dan mahasiswa SI memahami pola persebaran asal sekolah secara visual.

01 Dashboard Analitik

Ringkasan total mahasiswa, angkatan aktif, jalur dominan, sekolah dominan, dan tren terbaru dalam satu tampilan.

02 Peta Persebaran

Marker dan heatmap membantu membaca konsentrasi daerah asal mahasiswa secara cepat.

03 Filter Data

Filter angkatan, jalur penerimaan, dan pencarian sekolah membuat eksplorasi data lebih terarah.

04 Administrasi Data

Import Excel, geocoding otomatis, input manual, pratinjau data, dan export PDF untuk kebutuhan pelaporan.
